Estructuras Condicionales Compuestas

En JavaScript, las estructuras condicionales compuestas se utilizan para ejecutar un bloque de código si se cumple una condición, y otro bloque de código si la condición no se cumple. La estructura condicional compuesta tiene la siguiente sintaxis: Por ejemplo, el siguiente código comprueba si un número es mayor que 10. Si el número es mayor que 10, se imprime un mensaje en la consola. Si el número no es mayor que 10, se imprime otro mensaje en la consola.

Estructuras Condicionales Anidadas

Decimos que una estructura condicional es anidada cuando por la rama del verdadero o el falso de una estructura condicional hay otra estructura condicional. De esta manera nos queda una condición dentro de otra.
